✨ 일괄 변환에 한글(HWP) 파일 등록 가능하게

- 일괄 변환 로직(_batch_file_kind/_batch_target_extension/_extract_batch→_convert_hwp_file, 드롭·폴더 추가)은 이미 hwp 지원했으나, '파일 추가' 다이얼로그 필터에 .hwp/.hwpx가 빠져 선택이 안 됐음.
- add_batch_files 필터 '지원 문서'에 *.hwp;*.hwpx 추가 + '한글' 전용 필터 추가.
- 일괄 탭 안내 문구 'HWP는 제외됩니다' → 'PPT, Excel, Word, 한글(HWP), PDF, TXT' (PDF도 빠져 있던 것 함께 정정). 메인 안내에도 한글 추가.
- 한글 변환은 _convert_hwp_file 경로라 보안 접근 대화상자 Enter 자동통과(watcher)도 일괄에 그대로 적용.
