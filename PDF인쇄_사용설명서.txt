================================================================================
                    문서 PDF 인쇄 도구 - 사용 설명서
================================================================================

[프로그램 소개]
-------------------------------------------------------------------------------
회사 DRM(문서 보안)이 적용된 문서를 PDF로 인쇄하여 보안 없는 파일로 만드는 도구입니다.
프린터 선택이 잠겨있어도 강제로 PDF 프린터를 사용합니다.


[지원 문서]
-------------------------------------------------------------------------------
  - Microsoft PowerPoint (.pptx, .ppt)
  - Microsoft Excel (.xlsx, .xls)
  - 한글 (.hwp, .hwpx)


[작동 원리]
-------------------------------------------------------------------------------
1. 기본 프린터를 "Microsoft Print to PDF"로 임시 변경
2. 문서를 PDF로 인쇄 (프린터 선택 UI 우회)
3. 원래 기본 프린터로 복원

※ 프린터 선택이 잠겨있어도 기본 프린터는 변경 가능한 점을 활용합니다.


[사용 방법]
-------------------------------------------------------------------------------
1. 보안이 걸린 문서를 먼저 엽니다 (PowerPoint, Excel, 또는 한글)

2. PDFPrinter.exe를 실행합니다

3. 프로그램이 자동으로 열린 문서를 감지합니다
   - "PDF 프린터 상태"가 녹색으로 ✅ 표시되어야 합니다
   - 문서명과 종류가 표시됩니다

4. [찾아보기] 버튼을 클릭하여 PDF 저장 위치를 선택합니다

5. [PDF로 인쇄] 버튼을 클릭합니다

6. 인쇄가 완료되면 알림이 표시됩니다


[중요 주의사항] ★★★
-------------------------------------------------------------------------------
1. 저장된 PDF를 이 PC에서 열지 마세요!
   - 열면 DRM이 다시 암호화할 수 있습니다

2. USB, 외장하드, 네트워크 드라이브 등으로 복사 후 다른 PC에서 사용하세요

3. 인쇄 중에는 문서를 조작하지 마세요


[문제 해결]
-------------------------------------------------------------------------------
Q: "Microsoft Print to PDF 없음"이라고 표시됩니다
A: Windows 설정에서 기능을 활성화해야 합니다:
   1. Windows 설정 → 앱 → 선택적 기능
   2. "기능 추가" 클릭
   3. "Microsoft Print to PDF" 검색 후 설치

Q: "열린 문서 없음"이라고 표시됩니다
A: 문서를 먼저 열고 [다시 감지] 버튼을 클릭하세요

Q: 인쇄 후 PDF가 생성되지 않습니다
A: 저장 경로에 한글이나 특수문자가 있으면 영문 경로로 변경해보세요

Q: "PDF 프린터 설정 실패" 오류가 발생합니다
A: 관리자 권한으로 실행해보세요 (우클릭 → 관리자 권한으로 실행)


[PDF 활용 방법]
-------------------------------------------------------------------------------
저장된 PDF는 다음과 같이 활용할 수 있습니다:

1. 그대로 사용
   - PDF 뷰어로 열람
   - 인쇄, 배포

2. PPT로 변환
   - Adobe Acrobat: PDF → PowerPoint 내보내기
   - 온라인 변환: smallpdf.com, ilovepdf.com 등

3. Excel로 변환
   - Adobe Acrobat: PDF → Excel 내보내기
   - 온라인 변환: smallpdf.com, ilovepdf.com 등

4. 편집
   - Adobe Acrobat Pro로 직접 편집
