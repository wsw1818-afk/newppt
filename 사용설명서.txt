                    문서 보안 해제 저장 도구 - 사용 설명서
================================================================================

[프로그램 소개]
-------------------------------------------------------------------------------
회사 DRM(문서 보안)이 적용된 Office 문서를 정상 OpenXML 파일로 저장하거나,
TXT를 새 파일로 복사/변환하는 도구입니다.
PowerPoint, Excel, Word, 메모장을 지원합니다.


[지원 문서]
-------------------------------------------------------------------------------
  - Microsoft PowerPoint (.ppt, .pptx, .pptm, .ppsx, .potx)
  - Microsoft Excel (.xls, .xlsx, .xlsm, .xlsb)
  - Microsoft Word (.doc, .docx, .docm)
  - 텍스트 파일 (.txt)


[사용 방법]
-------------------------------------------------------------------------------
1. 보안이 걸린 문서를 먼저 열거나, 프로그램에서 [파일 선택]으로 원본 파일을 고릅니다.
   - PPT/Excel/Word/TXT 파일은 각 파일 선택 칸에 드래그앤드롭해도 됩니다.
   - 여러 파일이나 폴더는 [일괄 변환] 목록에 드래그앤드롭할 수 있습니다.

2. DocumentExtractor.exe를 실행합니다

3. 프로그램이 자동으로 열린 문서를 감지합니다
   - 문서명과 종류가 표시됩니다
   - 감지되지 않으면 [다시 감지] 버튼을 클릭하세요

4. [찾아보기] 버튼을 클릭하여 저장할 위치와 파일명을 선택합니다

5. [새 파일로 저장] 버튼을 클릭합니다

6. 저장이 완료되면 알림이 표시됩니다


[작동 원리]
-------------------------------------------------------------------------------
DRM은 Office의 "저장" 기능을 감시하지만, 이 프로그램은:

  1차 시도: SaveCopyAs 명령 (다른 이름으로 복사본 저장)
            → 성공하면 원본과 100% 동일한 파일 생성

  2차 시도: (1차 실패 시) Office 내부 구조 복원 또는 재구성
            - PowerPoint: 슬라이드 패키지 복원 또는 슬라이드 복제
            - Excel: 시트/셀/도형 정보를 새 Excel에 재구성
            - Word: WordOpenXML 구조 복원 또는 텍스트 재구성
            - TXT: 텍스트 복사 또는 DOCX 변환


[주의 사항]
-------------------------------------------------------------------------------
1. 열린 문서 방식과 파일 선택 방식을 모두 지원합니다
   - 열린 문서를 추출하려면 프로그램 실행 전에 문서를 열어주세요
   - 파일 선택을 쓰면 문서를 미리 열지 않아도 됩니다
   - 드래그앤드롭은 파일 선택 방식으로 처리됩니다

2. 여러 문서가 열려 있으면 콤보박스에서 처리할 문서를 선택하세요

3. 2차 방식(추출)으로 저장된 경우
   - PowerPoint/Excel/Word의 일부 복잡한 서식, 매크로, OLE 개체는 달라질 수 있음
   - 텍스트 재구성 방식은 .docx 저장만 지원함

4. 한글/HWP는 회사 DRM 환경에서 일반 파일 변환이 불가해 제외되었습니다

5. 저장 경로는 보안이 적용되지 않는 폴더를 선택하세요
   - USB 드라이브
   - 개인 폴더
   - 외부 네트워크 드라이브 등


[문제 해결]
-------------------------------------------------------------------------------
Q: "열린 문서 없음"이라고 표시됩니다
A: 문서를 먼저 열고, [다시 감지] 버튼을 클릭하세요

Q: 저장 버튼을 눌러도 반응이 없습니다
A: 저장 경로를 먼저 선택해주세요

Q: "SaveCopyAs 실패"라고 표시됩니다
A: DRM이 차단한 것입니다. 자동으로 2차 방식으로 시도합니다

Q: 저장된 파일이 깨져 보입니다
A: 2차 방식의 한계입니다. 1차 방식이 차단된 경우 발생할 수 있습니다

Q: HWP/한글 파일을 변환할 수 있나요?
A: 현재 버전에서는 지원하지 않습니다. 회사 DRM 컨테이너로 다시 저장되는 문제가 있어 제외했습니다.
