================================================================================
                    문서 보안 해제 저장 도구 - 사용 설명서
================================================================================

[프로그램 소개]
-------------------------------------------------------------------------------
회사 DRM(문서 보안)이 적용된 문서를 보안 없는 새 파일로 저장하는 도구입니다.
PowerPoint, Excel, 한글 문서를 지원합니다.


[지원 문서]
-------------------------------------------------------------------------------
  - Microsoft PowerPoint (.pptx)
  - Microsoft Excel (.xlsx)
  - 한글 (.hwp, .hwpx)


[사용 방법]
-------------------------------------------------------------------------------
1. 보안이 걸린 문서를 먼저 엽니다 (PowerPoint, Excel, 또는 한글)

2. DocumentExtractor.exe를 실행합니다

3. 프로그램이 자동으로 열린 문서를 감지합니다
   - 문서명과 종류가 표시됩니다
   - 감지되지 않으면 [다시 감지] 버튼을 클릭하세요

4. [찾아보기] 버튼을 클릭하여 저장할 위치와 파일명을 선택합니다

5. [새 파일로 저장] 버튼을 클릭합니다

6. 저장이 완료되면 알림이 표시됩니다


[작동 원리]
-------------------------------------------------------------------------------
DRM은 Office의 "저장" 기능을 감시하지만, 이 프로그램은:

  1차 시도: SaveCopyAs 명령 (다른 이름으로 복사본 저장)
            → 성공하면 원본과 100% 동일한 파일 생성

  2차 시도: (1차 실패 시) 내용을 직접 추출하여 새 파일 생성
            - PowerPoint: 슬라이드를 이미지로 내보내기 → 새 PPT에 삽입
            - Excel: 셀 데이터 직접 읽기 → 새 Excel에 복사
            - 한글: 전체 선택 후 복사 → 새 문서에 붙여넣기


[주의 사항]
-------------------------------------------------------------------------------
1. 문서를 먼저 열어야 합니다
   - 프로그램 실행 전에 보안 문서를 열어주세요

2. 한 번에 하나의 문서만 처리됩니다
   - 여러 문서가 열려있으면 활성화된 문서가 선택됩니다

3. 2차 방식(추출)으로 저장된 경우
   - PowerPoint: 이미지 기반이므로 텍스트 편집 불가
   - Excel: 수식이 값으로 변환될 수 있음
   - 한글: 대부분의 서식 유지됨

4. 저장 경로는 보안이 적용되지 않는 폴더를 선택하세요
   - USB 드라이브
   - 개인 폴더
   - 외부 네트워크 드라이브 등


[문제 해결]
-------------------------------------------------------------------------------
Q: "열린 문서 없음"이라고 표시됩니다
A: 문서를 먼저 열고, [다시 감지] 버튼을 클릭하세요

Q: 저장 버튼을 눌러도 반응이 없습니다
A: 저장 경로를 먼저 선택해주세요

Q: "SaveCopyAs 실패"라고 표시됩니다
A: DRM이 차단한 것입니다. 자동으로 2차 방식으로 시도합니다

Q: 저장된 파일이 깨져 보입니다
A: 2차 방식의 한계입니다. 1차 방식이 차단된 경우 발생할 수 있습니다

Q: 한글 문서가 감지되지 않습니다
A: 한글이 최신 버전인지 확인하세요. 일부 구버전은 지원되지 않습니다
